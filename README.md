# Локальный ИИ-ассистент Gemma (Ollama + Open WebUI)

## Требования:
### Программные:
- Docker
- Веб-браузер

### Аппаратные:
- ОЗУ: от 8 ГБ (для моделей 4b), от 16 ГБ (для 12b-27b)
- Процессор: Не говно
- Видеокарта (опционально, но рекомендуется): NVIDIA с поддержкой CUDA + установленный NVIDIA Container Toolkit.

## Первый запуск:
1. Билд контейнера:
```bash
docker compose up -d
```
2. Необходимо выбрать модель, с которой будешь работать: [тык](./docs/MODEL_CHOOSE.md)
3. После билда контейнера выполнить команду, где последний атрибут это версия выбранной модели
```bash
docker compose exec ollama ollama pull gemma3:4b
```
4. Перейти в браузер по адресу http://localhost:3000.
При первом входе создай учетную запись
> Первый пользователь автоматически получит права администратора.

## Повторный запуск:
```bash
docker compose up -d
```
## Остановка работы модели:
```bash
docker compose down
```
## Смена модели:
```bash
docker compose exec ollama ollama pull <имя_новой_модели>
```
